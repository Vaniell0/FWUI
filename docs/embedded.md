# Embedded Pages (constexpr)

## Концепция

`fwui-embed` рендерит HTML при сборке и бейкает результат в C++ исходник как `constexpr uint8_t[]`. Финальный бинарник содержит все страницы внутри — без файловой системы, без runtime-парсинга.

Идеально для роутеров, IoT-панелей, промышленных контроллеров — везде, где нет файловой системы или она ненадёжна.

## Сборка

```bash
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build --target fwui-embed
```

## Использование

```bash
./build/fwui-embed [options]
```

| Флаг | Описание | По умолчанию |
|------|----------|-------------|
| --output, -o <dir> | Директория для сгенерированных файлов | . |
| --pages <dir> | Шаблонные страницы | pages/ |
| --data <dir> | JSON-данные | data/ |
| --templates <dir> | Шаблоны (include) | templates/ |
| --minify | Минификация (по умолчанию) | true |
| --no-minify | Красивый HTML | — |

## Что генерируется

`fwui-embed` создаёт два файла:

### embedded_pages.hpp
```cpp
// Auto-generated by fwui-embed. Do not edit.
#pragma once
#include <fwui/embedded.hpp>
```

### embedded_pages.cpp
```cpp
// Auto-generated by fwui-embed. Do not edit.
#include "embedded_pages.hpp"

namespace fwui::embedded {

static constexpr uint8_t page_index[] = {
    0x3c,0x21,0x44,0x4f,0x43,0x54,0x59,0x50, // <!DOCTYP
    0x45,0x20,0x68,0x74,0x6d,0x6c,0x3e,...     // E html>...
};

static constexpr uint8_t page_about[] = { ... };

static constexpr PageData all_pages[] = {
    { "/", page_index, sizeof(page_index), false },
    { "/about", page_about, sizeof(page_about), false },
};

const PageData* FindPage(std::string_view route) { /* linear search */ }
size_t PageCount() { return sizeof(all_pages) / sizeof(all_pages[0]); }
std::span<const PageData> AllPages() { return all_pages; }

} // namespace fwui::embedded
```

## PageData (embedded.hpp)

```cpp
struct PageData {
    std::string_view route;      // "/about"
    const uint8_t* data;         // указатель на constexpr массив
    size_t size;                 // размер в байтах
    bool compressed;             // флаг сжатия (зарезервирован)

    std::string_view Content() const;  // data как string_view
};
```

### API

```cpp
const PageData* FindPage(std::string_view route);  // nullptr если не найдено
size_t PageCount();                                  // количество страниц
std::span<const PageData> AllPages();               // все страницы
```

## Использование в продакшене

Сгенерированные файлы линкуются в ваш бинарник. Работает с любым HTTP-сервером (Crow, Beast, httplib, Pistache и т.д.):

```cpp
#include "embedded_pages.hpp"

// Пример обработчика запроса (псевдокод)
void handle_request(const std::string& path, HttpResponse& resp) {
    auto* page = fwui::embedded::FindPage(path);
    if (page) {
        resp.set_header("Content-Type", "text/html");
        resp.set_body(std::string(page->Content()));
        resp.set_status(200);
    } else {
        resp.set_status(404);
    }
}
```

Нулевой runtime: `FindPage` → линейный поиск по constexpr массиву → `Content()` → reinterpret_cast → memcpy в ответ. Никакого чтения файлов, парсинга шаблонов, выделения памяти.

## Роутеры / IoT

Бинарник `fwui-embed` + ваш main.cpp = один исполняемый файл ~100-500 KB (зависит от количества страниц). Без runtime, без интерпретатора, без файловой системы. Подходит для:

- OpenWrt (роутеры, 8-32 MB RAM)
- ESP32 / STM32 (микроконтроллеры)
- Промышленные HMI-панели
- Appliance web UI
